QUICK START - Devfest-2026
========================================

========================================
OPTION 1: DOCKER (Recommended)
========================================

Prerequisites:
- Docker Desktop installed and running
- Copy env.example to .env and add your API keys

Commands:

# Build and start all services
docker-start.bat up

# Or manually:
docker-compose up -d

# Pull the Ollama model (first time only)
docker-start.bat pull-model

# View logs
docker-start.bat logs

# Stop all services
docker-start.bat down


========================================
OPTION 2: BATCH SCRIPT (Windows)
========================================

Prerequisites:
- Python 3.11+ installed
- Node.js 18+ installed
- WSL with Ollama installed

Just run:
start_all.bat

This will start everything automatically!

========================================
TROUBLESHOOTING:
========================================

Docker not starting?
  - Make sure Docker Desktop is running
  - Run: docker info

Missing API keys?
  - Copy env.example to .env
  - Add your GROQ_API_KEY

LinkedIn not working?
  - Create session first: python linkedin_scraper/samples/create_session.py
  - Mount linkedin_session.json in Docker
